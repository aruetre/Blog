HTML
Hypertext Markup Language, el lenguaje estándar para documentos diseñados para ser mostrados en un navegador web.
CSS
Cascading Style Sheets, un lenguaje de hojas de estilo usado para describir la presentación de un documento escrito en HTML o XML.
JavaScript
Un lenguaje de programación interpretado que se utiliza principalmente en páginas web para mejorar la interactividad con el usuario.
